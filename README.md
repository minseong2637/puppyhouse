> **프로젝트:** 개인 포트폴리오 사이트
>
> **기획 및 제작:** 김민성
>
> **분류:** 팀 프로젝트
>
> **제작 기간:** 2022.11 ~ 12.

>
> **사용 기술:** springboot, photoshop, dreamweaver, oracle
